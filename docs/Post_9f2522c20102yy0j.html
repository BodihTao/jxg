因果“1+1”与因果“微积分”
==================
（一）评论：甲骗了乙的钱，或甲赌博赢了乙的钱，应该是乙消业增福，但现实是有人因赌博输的倾家荡产而自杀，去年不是有个大学生被骗了学费而自杀吗？期待果师兄能解释一下。
========
吉祥观点：本文的标题，意思就是用1+1=2代表简单的因果，用微积分代表庞大复杂的系统性因果，是这么个概念。我有时候写东西不一定必须有条理，但是必须有道理。所以，我就随意写下下面的文字：
====
（1）、任正非在43岁被骗了200多万，王健林1993年被骗好几千万，柳传志创业之初被骗300万，你们可以去查一查，可是他们被骗却后来越来越强。但是你说的大学生被骗就自杀了，，这说明什么？说明同样被骗，因果不同，故而结局不同。

（2）、你说有人因赌博输的倾家荡产而自杀，可是还有很多人因赌博输的倾家荡产而没有自杀的。故而同样赌博，因果不同，结局不同。

（3）、赌博离不开高利贷，你怎么知道他赌输的钱必然一定是他自己辛辛苦苦赚来的，而不是他高利贷来的呢？也就是说，即便除开高利贷利息，你怎么确定他没有欠下巨额债务招致损福而死呢？

（4）、能够爱好赌博的人，首先先天带有“不劳而获”的思想基因，或者佛教叫做等流习气，不然他绝不会赌博，这你承认吧？因此，你又怎么判断被他赌输的家产不是之前通过其他“不劳而获”的手段获取的呢？劫夺而得之财物，必因劫夺而失之。得易失易，得难失难。

（5）、我邻居中就有老子贪污，而儿子赌博，拜光家业的两位，其中一位还是我以前的老领导。

（6）、我父亲幼时的玩伴中，有一老伯，其子与地方官员勾结，通过虚假项目骗取中央财政转移支付给地方上的大额补贴，之后也是厄运连连。这是老伯亲口对我讲的。我并不能确定他是否后来参与赌博，但是如果他后来参与了并且钱赌输了，如果自杀了。别人只会看见他赌博输钱自杀了，而不会追查他的财富来源是骗补所得，因此就认为他死得很冤，可是他到底冤不怨呢？就像我另外两个邻居，儿子输了家产，却没人追问他老子怎么搞来的这些钱？对不对？我因为非常欣赏这老伯儿子的书法，所以曾经留意过他，不然可能我很难记得这件事情。


（7）、你们知道那个“连环过桥贷”，那个最初的资金来源，严格意义上讲应该是银行出来的，但是实际情况却是，很多资金的本初的来源是不明不白的，并不都是银行出来的，其本初的资金来源，实际是一些有背景的人在冼黑钱，只是很多人觉察不到。所以，当一笔资金被层层盘剥十几次后，最后得到资金者居然携款逃跑了，而所有在这个链条上的参与者，都同时成为了债务人和债权人。这是这些年最普遍最经典的债务模式，包括后来大量P2P平台，也没有逃出这种模式，实际上P2P最后演变成了P2P2P2P2P2P2P模式，也就是我概括的“连环狗臭P”，这就是P2P。你们专业搞P2P的，你们看看我形容你们是不是非常贴切？因此，我之前安慰一些佛友的时候，不是曾经说过：有些债务是不用还的吗？为什么不用还呢？因为本初那个资金的来源就是黑金，但是黑金经过“连环狗臭P2P”，钱就漂白了。因此，因果注定这笔财富是回不去的，黑金本初得之非理非法，最终失之亦非理非法。有些人说我的讲法不道德，不遵守法律，不这个，不那个，，我告诉你，，遵守什么都超不过遵守因果的定律，因果注定还不了，就是还不了，你再对抗因果也还不了。你们要是不相信，你们可以用整个二十一世纪未来八十年的时间来验证我最近五年的判断，凡是最近二十年之内类似模式的债务，最后都是不了了之，都是还不了。信不信，让时间去验证。因为四十年来，社会领域大量坑蒙拐骗，正府领域大量摊乌受会，这些非法所得，必须有一个非法所失的管道，这就是因果。记住我这句经典评语。因果注定会非法失去的东西，你还想对抗因果，这是可笑的。大家看到的都是一个点的因果，甚至不一定看透，而我经常看到是一个面的因果，或者类似立体几何的因果，或者类似函数乃至微积分一般的因果。你只看一个点的因果，当然看不透。当然，在这个复杂链条上每一个债务人和债权人，你应该思考的是，资金在你的这个环节上，你从里面拿走了多少钱，这才是你的债务问题，至于从你手里留下下一个P的资金，那个就和你无关了。我们透过这么复杂的分析，不外乎是告诉你：你所见到的“赌博倾家荡产导致自杀现象”只是复杂的因果链条上的一个细微的环节而已，你要看透所有的环节，你才知道他为什么自杀。


（8）、赌博导致妻离子散，本身就抽走家庭福德，导致家庭系统趋向封闭，能量总量减少，导致熵增迅速达到上限，这也是你没有考虑的。换句话说，家庭中的福田没了，你想找个人付出，都没人接受你付出，你想转移一点福报给自己，连接受你转移福报的对象都找不到。


（9）、更何况，做善事的人尚且也有自杀而死的。为什么经常赌博的人就不能自杀而死？由此可见，他死一定有其他方面福报耗尽的原因，有其他方面恶报成熟的原因，未见得必然只是因为赌博这一个因缘。



======================
（二）谈一谈奴隶制与资本主义制度
======================


评论：（A）、不知道该怎么评价了，照果师兄的理论，我们当初就不该解放西藏那些农奴！因为他们不知道前几世都是欠压榨剥削他们的农奴主的。（B）、林肯解放农奴难道错误了？
========

吉祥观点：（1）、首先你说的这种状况不是可能性，而是存在必然性，但不一定是农奴主与农奴身份互换来代表前世今生，也可能是人和动物，或者动物和动物的关系。比如可能你过去豢养动物没有自由，盘剥对待动物，想打就打，想杀就杀，最后转世为奴隶，而被动物转世的奴隶主这样对待，也未可知。

（2）、该被压迫就有被压迫的因果，被压迫之业报受完了，自然就会有人来解放被压迫者，谁知道解放农奴的一方，不是宿世被放生而解放的动物报恩而来呢？对不对？所以，解放有解放的因果，压迫有压迫的因果。我的观点：今受奴隶果，宿因奴人是，来世奴隶果，今生奴人是。

（3）、我晓得你对我替财阀资本家说话的理论不满，但是我说一句戳心的话，是什么让你成为了奴隶？除了业力，还有什么？为什么那么多汉人怎么很早就结束奴隶制，为什么西藏那么晚？那不是代表他们的业报没有结束，还能代表什么呢？

（4）、另外，你用西藏奴隶制来揶揄我对资本家的评断，并不合理。请你分清楚：资本家和劳动者的雇佣关系是双方自愿认可的，而奴隶主和奴隶的关系是奴隶主之间认可而奴隶自身无权决定的。这是二者的不可比性。资本家和劳动者间是自愿的交易关系，如果你认为这个交易是不公平的剥削，任意一方可以在契约下自由终止雇佣关系，比如最近曹德旺就说了，如果镁国工会敢于入驻福耀玻璃镁国分厂，他立即撤资退出在美投资。但是奴隶和奴隶主之间不是这种关系，第一奴隶是不自愿被强迫的，第二奴隶是没有报酬的，第三奴隶主可以随时结束奴隶的生命。这三条都和资本家不一样。所以，你认为从这三点来看，西藏的奴隶制可以和我所讲的资本雇佣关系相提并论吗？如果可以相提并论，马恩理论对社会阶段的划分就被你推倒了！因此，你用奴隶主类比资本家，根本不合理。你给资本家干活，受了伤，资本家还要赔偿，你给奴隶主干活，受了伤，资本家看你以后干不了活了，养着也会白吃粮食，直接就把你生命结束了，对不对？怎么比呢？

（5）、我明白你要表达什么：资本家如果对工人压迫怎么办？你应该听说过大罢公吧？你应该听说过辞职吧？现在的资本家敢拿鞭子抽你吗？有些人丑化资本家的时候，动不动喜欢说资本家拿鞭子抽人，记住，那是奴隶主干的活，不是资本家干的活。
